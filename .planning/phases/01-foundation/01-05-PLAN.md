**Step 3: Set NEXT_PUBLIC_BASE_URL environment variable**

Once the project is created:
1. Go to your Vercel project dashboard
2. Click "Settings" → "Environment Variables"
3. Add the variable:
   - **Name:** `NEXT_PUBLIC_BASE_URL`
   - **Value for Production:** your production domain (e.g. `https://astraos.com` or the Vercel-assigned URL like `https://astra-lp.vercel.app`)
   - **Value for Preview:** your Vercel preview base URL (e.g. `https://astra-lp-git-main-yourname.vercel.app`)
   - **Value for Development:** `http://localhost:3000`
4. Save. Redeploy the project so the variable takes effect.

**Step 4: Note the preview URL**

From the Vercel dashboard → Deployments, copy the preview deployment URL. You will need this when configuring LinkedIn OAuth callback URLs in Phase 3.
